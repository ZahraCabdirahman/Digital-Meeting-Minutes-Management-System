# Digital Meeting Minutes Management and Collaboration System

Digital Meeting Minutes Management and Collaboration System is a web-based platform for managing the full meeting workflow from scheduling to minutes recording, participant collaboration, task assignment, document handling, notifications, and reporting.

Live system: https://digitalmeeting24.com/

## System Overview

The system helps organizations replace manual meeting records with a secure digital process. Administrators and meeting organizers can schedule meetings, invite participants, define agendas, record structured minutes, assign follow-up tasks, upload meeting documents, and generate meeting reports. Participants can view their assigned meetings, access minutes and documents, submit tasks, and update their profile from a dedicated participant dashboard.

The platform includes authentication and role-based access control so that users only access the areas and actions allowed for their role.

## Features

- User authentication and session management
- Role-based access for Admin, Meeting Organizer, and Participant users
- User management and profile management
- Meeting scheduling with agenda, date, time, location, and participants
- Structured meeting minutes recording
- Discussion points, decisions, and action item tracking
- Meeting minutes version history and restore support
- Participant dashboards for meetings and assigned tasks
- Task assignment, progress tracking, submission, approval, and rejection
- Meeting document upload and download
- Email notifications for meetings, minutes, and tasks
- Real-time collaboration and comments using Socket.IO
- Dashboard analytics and meeting reports
- Secure API structure for frontend and backend communication
